Nim Game — Computational Thinking
An interactive Nim game designed for use on an electronic whiteboard in computer science lessons. Built as a single self-contained HTML file — no installation required.
What is Nim?
Nim is a two-player strategy game. Players take turns removing one or more counters from a single pile. The player who takes the last counter wins.
Teaching Features

Hint mode — reveals the binary representation of each pile, allowing the teacher to introduce the concept of binary numbers in a game context
Show Nim-Sum — displays the XOR of all pile values, helping students understand the winning strategy
Suggest — highlights the optimal move and, when Hint mode is also on, explains step-by-step why it is the best move using binary arithmetic
Autoplay — either or both players can be set to play automatically. The computer always plays the winning strategy when one exists, and plays randomly when it doesn't — useful for demonstrating what happens when the strategy breaks down

How to Play
Tap counters to select them (selected counters turn green). Press Take to complete your turn. The game tracks wins across multiple rounds.
Setup
Use the Setup button to configure player names, the number of piles, and the number of counters per pile. Several presets are included for quick classroom use.
Hosting
This is a static single-file app. It can be hosted on GitHub Pages or any web server, or simply opened locally in a browser.
